# Bus-Pass-Management-System
Bus Pass Management System built using HTML, CSS, JavaScript, Node.js, and MySQL. Users can apply for new passes, renew existing ones, and manage details online. This project digitalizes the bus pass process and is developed for learning and academic full-stack web development

## Overview
The **Bus Pass Management System** is a web-based application developed as a mini project using **HTML, CSS, JavaScript (Frontend)** and **Node.js + MySQL (Backend)**. This system allows users to apply for new bus passes, renew existing passes, and manage their personal details online. It helps digitalize the bus pass process, reducing paperwork and manual effort. The project demonstrates basic full-stack web development concepts in a practical way.

## Features
- **User Registration & Login** – Secure login for users and admins.  
- **Apply for Bus Pass** – Users can submit applications online.  
- **Renew Pass** – Existing users can renew their passes easily.  
- **Admin Panel** – Admin can view, approve, or reject applications.  
- **Database Integration** – All user and pass details are stored in MySQL.  
- **Responsive Design** – Works on desktop and mobile devices.  

## Technologies Used
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express.js  
- **Database:** MySQL  
- **Version Control:** Git & GitHub  

## How It Works
1. Users visit the website and register with their details.  
2. They can apply for a new bus pass or renew an existing one.  
3. Admin logs in to the admin panel to approve or reject applications.  
4. All information is securely stored in the MySQL database.
